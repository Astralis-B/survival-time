## [1.0.0] - 2026-02-25
### Initial Release
- **Added**: Survival time tracker — displays time elapsed since the last death
- **Added**: Two time formats:
  - **Real Time** — decimal real-world units (Seconds → Minutes → Hours → Days → Months → Years)
  - **Minecraft Days** — decimal in-game units (Hours → Days → Months → Years)
- **Added**: Death animation — surviving time counts down to zero on death, then fades out over 8 seconds
- **Added**: 6 configurable HUD positions: Top Left, Top Right, Bottom Left, Bottom Right, Action Bar, Center Top
- **Added**: X and Y pixel offset controls for fine-tuned placement
- **Added**: ModMenu + Cloth Config integration for full in-game configuration
- **Added**: Client-side only — no server mod required
